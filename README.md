🤖 How to Access Claude (AI) in VS Code
Claude is Anthropic's AI assistant. Here are the most popular ways to use it directly inside VS Code:

**Option 1 – GitHub Copilot (Claude models)**
1. Install the **GitHub Copilot** extension from the VS Code Marketplace.
2. Sign in with your GitHub account (a Copilot subscription is required).
3. Open the Copilot Chat panel and click the model-selector dropdown.
4. Choose a Claude model (e.g. *Claude 3.5 Sonnet*) from the list.
5. Start chatting or using inline completions powered by Claude.

**Option 2 – Cline (Claude Dev) extension**
1. Open VS Code and go to the Extensions Marketplace (`Ctrl+Shift+X`).
2. Search for **Cline** and install it.
3. Open the Cline panel and select **Anthropic** as the provider.
4. Paste your **Anthropic API key** (get one free at https://console.anthropic.com).
5. Choose a Claude model (e.g. `claude-3-5-sonnet-20241022`) and start coding.

**Option 3 – Continue extension**
1. Install the **Continue** extension from the VS Code Marketplace.
2. Open the Continue config file (`~/.continue/config.json`).
3. Add Claude as a provider:
   ```json
   {
     "models": [
       {
         "title": "Claude 3.5 Sonnet",
         "provider": "anthropic",
         "model": "claude-3-5-sonnet-20241022",
         "apiKey": "YOUR_ANTHROPIC_API_KEY"
       }
     ]
   }
   ```
4. Save the file and reload VS Code. Claude will appear in the Continue chat panel.

> **Tip:** Keep your API key in a `.env` file and never commit it to version control.
